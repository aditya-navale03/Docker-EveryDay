# 1. docker pull IMAGE_NAME
--> to download new image of docker

# 2. docker images 
--> to see which images we have all

# 3. docker run IMAGE_NAME
--> to create container from any docker image.
--> we can directly run this command to automatically pull and create a container.

# 4. docker run -it IMAGE_NAME
--> to run in interactive mode.
--> allow us to access terminal which used to input or output something

# 5. doxker ps -a 
--> to see how many containers we have

# 6. docker ps 
--> to see running containers

# 7. docker start CONT_NAME or CONT_ID
--> to start a existing container

# 8. docker stop CONT_NAME or CONT_ID
--> to stop existing container

# 9. docker rmi IMAGE_NAME
--> to remove any docker image

# 10. docker rm CONT_NAME
--> to remove docker container
